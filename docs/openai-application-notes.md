## Project summary

Sleep Compass is a self-hosted, cross-device sleep dashboard. It imports iPhone Health sleep data via Health Auto Export and Google Drive, then displays it on web, PC, and Android. It helps users understand sync status, missing data, and sleep records for self-monitoring, not medical diagnosis.

## Use of OpenAI credits

If awarded OpenAI credits, I would use them to grow Sleep Compass from a personal sleep dashboard into a privacy-conscious, self-hosted health data viewer. I would add AI-assisted weekly summaries, sync troubleshooting, missing-data explanations, and gentle sleep/activity insights for self-monitoring, not medical diagnosis.

## Special note

Sleep Compass is not a medical diagnosis tool. It is a privacy-conscious, self-hosted way to make personal health data easier to understand across devices. It focuses on clarity: sync status, missing-data reasons, sleep-day boundaries, and gentle self-monitoring insights.
